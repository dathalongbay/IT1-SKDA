Bộ chọn ( Selector ) trong CSS
        ví dụ
        style cho thẻ p có chữ xanh
        style cho thẻ div có chữ đỏ
        style cho thẻ h1 có chữ vàng
        => Làm sao để chọn đúng cái đối tượng mình muốn style ?
        Câu trả lời : Sử dụng bộ chọn ( selector ) trong css
        Bộ chọn đơn giản nhất chính là tên thẻ html
        vi dụ muốn chọn thẻ p trong html thì sử dụng selector là p { code css }
        vi dụ muốn chọn thẻ div trong html thì sử dụng selector là div { code css }
        vi dụ muốn chọn thẻ h1 trong html thì sử dụng selector là h1 { code css }
        Sau đó sẽ viêt code css bên trong ngoặc nhọn {} theo sau selector
