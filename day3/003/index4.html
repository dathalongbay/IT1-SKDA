Sử dụng nhiều thuộc tính css trong code css theo dạng Internal Style
        Chú ý : Inline style ( dùng thuộc tính html style="" ) => viết code css trong cùng 1 dòng
        Chú ý : Internal style ( dùng thẻ style trong thẻ head ) => viết code css theo nhiều dòng khác nhau
        Chú ý : cuối mỗi cặp thuộc tính css : giá trị phải kết thúc bằng dấu ;
