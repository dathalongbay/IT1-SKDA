Thuộc tính background-repeat ( lặp lại ảnh nền khi kích cỡ khối thẻ
        lớn hơn so với kích thước hình ảnh )
        nếu kích thước hình ảnh lớn hơn kích thước của thẻ html thì thuộc tính
        background-repeat không có tác dụng

        Thuộc tính css background-repeat có 4 giá trị
        1 - repeat ( lặp lại theo cả chiều ngang và dọc ) ( mặc định )
        2 - no-repeat ( không lặp )
        3 - repeat-x ( lặp theo chiều ngang )
        4 - repeat-y ( lặp theo chiều dọc )
