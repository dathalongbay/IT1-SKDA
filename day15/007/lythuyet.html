cả 4 lệnh include, include_once , require_once , require
        đều dùng để nạp 1 file php vào 1 file khác
        sự khác nhau :
        - include và include_once : khi nạp file bị lỗi thì chương trình
        vẫn chạy tiếp
        - require và require_once : khi nạp file bị lỗi thì chương trình
        sẽ dừng luôn

        =>
        include sẽ nạp file nhiều lần khi mà bị gọi nhiều lần
        include_once sẽ nạp file 1 lần duy nhất khi mà bị gọi nhiều lần
