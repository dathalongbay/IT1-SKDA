Hướng đối tượng trong PHP
        class là lớp
        object là đối tượng
        muốn khởi tạo đối tượng từ class ta dùng từ khóa new
        ví dụ : $object1 = new class();
        extends là kế thừa
        khi 1 class con kế thừa từ class cha

        class bao gồm
        property ( thuộc tính )
        method ( phương thức )
        thuộc tính để chỉ tính chất đặc điểm của đối tượng
        còn phương thức để chỉ ra các hành động của đối tượng
