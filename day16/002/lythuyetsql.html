** chú ý các câu lệnh sql kết thúc bằng dấu ;
        Câu lệnh sql tạo cơ sở dữ liệu :
        CREATE DATABASE tên_cơ_sở_dữ_liệu;
        ví dụ :
        CREATE DATABASE opencart
        Câu lệnh sql xóa cơ sở dữ liệu
        DROP DATABASE tên_cơ_sở_dữ_liệu;
        ví dụ :
        DROP DATABASE opencart;
        Câu lệnh sql tạo bảng trong cơ sở dữ liệu :

        CREATE TABLE tên_bảng (
            tên_cột_1 kiểu_dữ_liệu(độ_dài) UNSIGNED AUTO_INCREMENT PRIMARY KEY,
            tên_cột_2 VARCHAR(30) NOT NULL,
            tên_cột_3 VARCHAR(30) NOT NULL,
            tên_cột_4 VARCHAR(50),
            tên_cột_5 TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
        )
        trong đó :
        UNSIGNED có nghĩa là không âm
        AUTO_INCREMENT có nghĩa là tự động tăng
        thông thường trong 1 bảng sẽ có 1 cột làm khóa chính
        khóa chính chứa các số nguyên tự động tăng dần và không trùng lặp giữa các bản ghi
        PRIMARY KEY là khóa chính
        NOT NULL là cho phép cột dữ liệu đó không bị NULL
        TIMESTAMP là 1 kiểu dữ liệu chứa thời gian trong mysql
        VARCHAR là 1 kiểu dữ liệu chứa chuỗi trong mysql
        INT là 1 kiểu dữ liệu số nguyên trong mysql
        DEFAULT là đặt dữ liệu mặc định cho cột dữ liệu khi người dùng không nhập
        CURRENT_TIMESTAMP là thời gian hiện tại trong mysql
        ON UPDATE là câu lệnh xử lý khi xảy ra sự kiện update của 1 bản ghi
        ví dụ :
        CREATE TABLE MyGuests (
            id INT(6) UNSIGNED AUTO_INCREMENT PRIMARY KEY,
            firstname VARCHAR(30) NOT NULL,
            lastname VARCHAR(30) NOT NULL,
            email VARCHAR(50),
            reg_date TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
        )
        thử gõ lại câu lệnh sau để tạo ra 1 bảng MyGuests
